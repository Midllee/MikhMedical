                                                            ПАТТЕРНЫ В ПРОЕКТЕ:
                        ==============---------------------------------------------------------=================
                                                                    MVC
                                                            Model-View-Controller

                        Model
                        Хороший пример Model будет класс Users, который используется для взаимодействия с базой данных.

                        View
                        Каждый метод в файле app.py возвращает шаблоны HTML

                        Controller
                        Контроллеры представлены функциями маршрутизации, которые обрабатывают запросы и данные
                        Например страница регистрации и страница входа в систему в моем проекте
